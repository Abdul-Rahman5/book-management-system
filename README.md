# Book Management System

A modern, responsive, and minimalist Book Management System built using React 19 and Vite. This application performs full CRUD operations integrated with a remote API, featuring dynamic real-time form validation and live client-side searching/filtering.

## 🚀 Live Demo
- **Live URL:** [https://book-management-system-two-gamma.vercel.app/]
## Setup Instructions:
1. Clone the repository: git clone https://github.com/Abdul-Rahman5/book-management-system.git
2. Install dependencies: npm install
3. Run the development server: npm run dev
4. Build for production: npm run build
   
## ✨ Key Features
- **Full CRUD Management:** Seamlessly View, Add, Edit, and Delete books.
- **Dynamic Dashboard (No Home Page):** Opens directly into the **Books List** catalog for a swift, data-driven user experience.
- **Real-time Form Validation:** Powered by **Formik** and **Yup** providing instant visual feedback (Bootstrap `is-invalid` states) without alert interruptions.
- **Instant Search & Filter:** Live client-side filtering by book **Title** or **Author**, plus a dynamic dropdown filter by **Genre**.
- **Robust API Client:** Centralized HTTP requests handled via **Axios** complete with custom loading indicators and global error state layout.
- **Clean Architecture:** Strict separation of concerns utilizing React **Context API** for state management and decoupled clean UI components.

## 🛠️ Tech Stack & Dependencies
- **Bundler / Environment:** Vite
- **Frontend Core:** React 19 & React DOM 19
- **Routing:** React Router DOM v7
- **Form Handling & Validation:** Formik & Yup
- **HTTP Client:** Axios
- **Styling & UI:** Bootstrap 5, FontAwesome, and Bootstrap Icons
